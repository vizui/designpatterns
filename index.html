What is this?

An open source design guide, pattern library, and Bootstrap theme for use in USPTO projects.

How do I use it?

Once downloaded from the link above or via Bower, you can use either the minified CSS build or the source LESS files in your own build system.


Who is it for?

This is intended for designers and developers working on projects related to the USPTO, but since it's public domain, others might find value in it as well.

Why did we make it?

We have a lot of web projects at the USPTO. These design guidelines and resources aim to create a unified experience across all projects at the USPTO, both internal and external.


Not using Bootstrap?

That's fine! A lot of our projects use Bootstrap, but not all of them. The design guidelines and some resources may still be useful (e.g., the icon font or brand LESS variables).

Want to contribute?

We want your help! Visit our Github for contribution guidelines. If you have improvements or bug fixes, let us know!
